1. Java Sources - top

Package com.brandcoder.*

Note: The package name will vary depending on the license version that you purchased

  • IntroActivity.java - This is the first Activity which will be started when the application is opened. It contains an image file to display to the users which presents your brand image at the launch of the app.
  • ChannelsActivity.java - This is the main Activity which handles the menu and channel display.
  • ProgressIndicator.java - This class handles the loading gif animation that is displayed while the channels are loading.

2. Resources - top

2.1. XML Files - top

Folder "layout"

  • main.xml - layout of ChannelsActivity
  • splash.xml - layout of IntroActivity
  • horslidingdrawer.xml - layout of the horizontal SlidingDrawer (menu that opens horizontally)
  • verslidingdrawer.xml - layout of the vertical SlidingDrawer (menu that opens vertically)
  • tabcontent.xml - layout of a single tab
  • tabimage.xml - layout of the image in a single tab
  • tabtext.xml - layout of the text in a single tab

Folder "values"

  • strings.xml - all text values of the application
  • colors.xml - the gradient colors of the loading bar that is displaying when some url is currently loading

2.2. Drawables - top

The drawable files are placed according to the device resolution in the following folders: drawable, drawable-mdpi, drawable-ldpi, drawable-hdpi, drawable-xhdpi.

  • openarrow.png - drawable with the arrow that opens the bottom sliding menu
  • closearrow.png - drawable with the arrow that closes the bottom sliding menu
  • horopenarrow.png - drawable with the arrow that opens the side sliding menu
  • horclosearrow.png - drawable with the arrow that closes the side sliding menu
  • splash.png - drawable with the image for the loading splash screen
  • tab_*.png - drawable with the image for one of the channel tabs in the menu, where * is the name of the tab
  • ic_launcher.png - icon of the application
  • ic_launcher_red.png - icon of the application in red color

3. Others Files - top

  • AndroidManifest.xml - the manifest of BrandCoder application
  • project.properties - properties of the BrandCoder application

III. How to Customize BrandCoder Mobile TV Android Application

1. Splash Screen - top

To customize the Splash Screen for your own application do the following:

  1. Replace the splash.png file in folders drawable, drawable-xhdpi, drawable-hdpi, drawable-mdpi and drawable-ldpi with your own splash image.

2. License Key - top

To set your License Key for your own application do the following:

  1. Replace the string constant in ChannelsActivity.java file in package com.brandcoder.* with your own license key.

3. Scrollable Channel Tabs - top

To add or modify a single Tab do the following:

  1. Add a new image for the tab icon in folder drawable.
  2. Add a new TabContent object in method getTabsContent() in ChannelsActivity.java file in package com.brandcoder.* as shown in the image below. The first parameter in red is the id of the resource of the image added in step 1. The second parameter is the text of the tab. The third one is the URL that is invoked when clicking on the tab.

To remove a single Tab do the following:

  1. Remove a TabContent object in method getTabsContent() in ChannelsActivity.java file in package com.brandcoder.*.

4. Enable AdMob Advertisement - top

AdMob advertisement is very easy to configure and once it is done the banners will ne shown in the top of the screen.To enable AdMob advertisement do the following:

  1. Set the boolean field isAdmobEnable in the file ChannelsActivity.java to true. By default the filed is false and the AdMob advertisement is not enabled. The field is shown in the bottom image.
  2. Fill your AdMob Publisher id in the field MY_AD_UNIT_ID. You can find your AdMob Publisher id in your setting page in the admob website. The field is shown in the bottom image.

5. Define the position of the tabs. - top

By default tabs are displayed in the bottom of the screen. If you like the way they are positioning you should not do anything. If you want to move them to the righ of the screen do the following:

  1. Change the value of the field sliderBottom in the file ChannelsActivity.java to false.The field is shown in the bottom image.

6. Define the gradient color of the top loading bar. - top

By default the gradient color is dark to light blue. If you want to change it to any other color open the colors.xml file which is located in the res/values folder in the project and do the following:

  1. Change the values of the colors as you like them to be displayed
