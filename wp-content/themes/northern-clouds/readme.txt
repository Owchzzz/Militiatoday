== Description ==
Northern-Clouds is a responsive HTML5/CSS3 theme. It uses an Ubuntu font. A horizontal version of 'Superfish' jQuery drop-down menu is built in, which supports touch devices. It supports custom headers and backgrounds. Translation ready.
